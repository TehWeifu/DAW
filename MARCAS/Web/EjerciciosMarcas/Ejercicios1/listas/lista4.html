Ejercicio11. Diccionario de términos

TCP/IP
Protocolo de redes de área extensa utilizado en la red de Internet.
Internet
Red de ordenadores de ámbito mundial.
HTML
HyperTextMarkup Language. Lenguaje etiquetado de hipertexto.
JavaScript
Lenguaje de programación de tipo script basado en Java. Es muy utilizado en Internet.
Host
Ordenador conectado a Internet
HTTP
Hypertext Transfer Protocol. Servicio de Internet que permite la transferencia de webs entre ordenadores.
LAN
Local Area Network. Red de área local.
