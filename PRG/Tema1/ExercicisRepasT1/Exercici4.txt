b) A > C 			→ false
c) A < C 			→ false
d) B < C 			→ false
e) B != C 			→ true
f) A == 3 			→ false
g) A * B == 15  		→ true
h) A * B == -30 		→ false
i) C / B < A			→ true
j) C / B == -10 		→ false
k) C / B == -4 		→ true
l) A + B + C == 5		→ false
m) (A+B == 8 ) && (A-B == 2)	→ true
n) (A+B == 8 ) || (A-B == 6) 	→ true
o) A > 3 && B > 3 && C < 3 	→ false
p) A > 3 && B >= 3 && C < -3 	→ true
